# 固有値問題とは何か？

固有値問題とは：

行列 (H) に対して

次を満たす状態を探す問題です：

```text
H |ψ⟩ = E |ψ⟩
```

ここで：

|ψ⟩ = 固有状態
E = 固有値

です。

---

# なぜ固有値が重要なのか？

理由：

物理の基本方程式：

```text
H |ψ⟩ = E |ψ⟩
```

が意味するのは：

エネルギーが決まる状態

を求めることだからです。

つまり：

固有値問題 = エネルギー問題

です。

---

# Hamiltonianと固有値

Hamiltonian の固有値は：

系のエネルギー

を表します。

例えば：

最小固有値：

```text
E₀
```

は：

基底状態エネルギー

です。

---

# 固有値問題とVQEの関係

VQEは：

次の問題を解いていました：

```text
min ⟨ψ|H|ψ⟩
```

これは：

最小固有値

を求める問題です。

つまり：

VQE = 最小固有値探索

です。

---

# 固有値問題と時間発展の関係

時間発展演算子：

```text
exp(-iHt)
```

を考えます。

もし：

```text
H |ψ⟩ = E |ψ⟩
```

なら：

```text
exp(-iHt)|ψ⟩ = exp(-iEt)|ψ⟩
```

になります。

つまり：

固有状態は

位相だけ変化します。

---

# ここが重要なポイント

重要な性質：

固有状態は

時間発展しても形が変わらない

ということです。

変わるのは：

位相だけ

です。

---

# 位相と固有値の関係

次の関係が成立します：

```text
|ψ(t)⟩ = exp(-iEt)|ψ⟩
```

つまり：

固有値 E は

位相変化速度

です。

---

# なぜこれが重要なのか？

理由：

位相を測定できれば

固有値が分かる

からです。

これが：

量子位相推定（QPE）

の基本アイデアです。

---

# 固有値問題の例（1量子ビット）

例：

Pauli-Z演算子：

```text
Z |0⟩ = +1 |0⟩
Z |1⟩ = −1 |1⟩
```

つまり：

固有値：

```text
+1
−1
```

固有状態：

```text
|0⟩
|1⟩
```

です。

---

# 2量子ビットの場合

例：

```text
H = Z₀Z₁
```

固有値：

```text
+1
−1
```

固有状態：

```text
|00⟩
|01⟩
|10⟩
|11⟩
```

です。

---

# なぜ固有値問題が難しいのか？

理由：

行列サイズが

指数的に増える

からです。

n量子ビットなら：

```text
2ⁿ × 2ⁿ 行列
```

になります。

---

# 古典計算での困難さ

例：

40量子ビット：

```text
2⁴⁰ ≈ 10¹²
```

つまり：

巨大行列になります。

対角化は困難です。

---

# 量子コンピュータの利点

量子コンピュータでは：

状態を直接扱えます：

```text
|ψ⟩
```

つまり：

行列全体を扱う必要がありません。

---

# 固有値問題の量子アルゴリズム

代表例：

量子位相推定（QPE）

です。

目的：

```text
H |ψ⟩ = E |ψ⟩
```

の

E

を求めることです。

---

# 位相推定の基本アイデア

時間発展：

```text
exp(-iHt)
```

を使うと：

```text
exp(-iEt)
```

が得られます。

つまり：

位相を測れば

固有値が分かります。

---

# VQEとQPEの違い

比較：

| 方法  | 特徴     |
| --- | ------ |
| VQE | 近似     |
| QPE | 高精度    |
| VQE | NISQ向き |
| QPE | FTQC向き |

つまり：

将来の主役は QPE

です。

---

理由：

指数的高速化

が可能だからです。

量子化学
材料科学
暗号

に重要です。

です。

固有値問題を解けば
