# 第02回 Pauli演算子によるHamiltonian表現

## はじめに

前回は：

* Hamiltonianとは何か
* なぜ演算子として表現されるのか
* 固有値問題との関係
* 相互作用Hamiltonian
* Isingモデル
* 時間発展との関係

を整理しました。

今回は：

Hamiltonianを

**量子回路として実装できる形に変換する方法**

を扱います。

その鍵となるのが：

**Pauli演算子分解**

です。

---

# なぜPauli演算子が重要なのか？

量子回路は：

基本的に

X
Y
Z

とそれらの組み合わせ

で構成されます。

つまり：

HamiltonianをPauli演算子で表現できれば

回路として実装できます。

---

# Pauli演算子とは何か？

Pauli演算子は：

最も基本的な量子演算子です。

3種類あります：

```text
X = [ 0  1 ]
    [ 1  0 ]

Y = [ 0 -i ]
    [ i  0 ]

Z = [ 1  0 ]
    [ 0 -1 ]
```

そして：

恒等演算子

も重要です：

```text
I = [ 1  0 ]
    [ 0  1 ]
```

---

# Pauli演算子の意味

それぞれ：

次の物理量に対応します：

X → x方向のスピン
Y → y方向のスピン
Z → z方向のスピン

つまり：

スピンの観測量

です。

---

# 任意のHamiltonianはPauli分解できる

重要な事実：

任意のHamiltonianは

Pauli演算子の線形結合として表現できます。

例えば：

```text
H = aX + bY + cZ
```

のように書けます。

さらに：

複数量子ビットでは：

```text
H = aX₀ + bZ₁ + cX₀Z₁
```

のようになります。

---

# なぜPauli分解できるのか？

理由：

Pauli演算子は

行列空間の基底

だからです。

つまり：

任意の2×2行列は

I, X, Y, Z

の線形結合として書けます。

---

# 2量子ビットの場合

2量子ビットでは：

基底は次の16個になります：

```text
I⊗I
I⊗X
I⊗Y
I⊗Z
X⊗I
X⊗X
X⊗Y
X⊗Z
Y⊗I
Y⊗X
Y⊗Y
Y⊗Z
Z⊗I
Z⊗X
Z⊗Y
Z⊗Z
```

つまり：

任意の4×4行列も

これらで表現できます。

---

# テンソル積とは何か？

複数量子ビットでは：

テンソル積

を使います。

例えば：

```text
Z₀Z₁
```

とは：

```text
Z ⊗ Z
```

の意味です。

これは：

2つの量子ビット間の相互作用

を表します。

---

# Pauli文字列とは何か？

次のような表現：

```text
X₀Y₁Z₂
```

を

**Pauli文字列**

と呼びます。

意味：

3量子ビット系の演算子

です。

---

# Hamiltonianの一般形

多くのHamiltonianは：

次の形で書かれます：

```text
H = Σ cᵢ Pᵢ
```

ここで：

cᵢ は係数
Pᵢ はPauli文字列

です。

これが：

Pauli分解されたHamiltonian

です。

---

# 例：1量子ビットHamiltonian

例：

```text
H = 0.5X + 0.8Z
```

これは：

2つのPauli演算子の線形結合です。

つまり：

2つの観測量の重ね合わせ

です。

---

# 例：2量子ビット相互作用

例：

```text
H = Z₀Z₁
```

意味：

2量子ビットの相互作用

です。

これは：

Isingモデルの基本項です。

---

# Ising Hamiltonianの一般形

Isingモデルは：

次の形を持ちます：

```text
H = Σ Jᵢⱼ ZᵢZⱼ + Σ hᵢ Zᵢ
```

ここで：

Jᵢⱼ は相互作用強度
hᵢ は外部磁場

です。

---

# なぜIsingモデルが重要なのか？

理由：

多くの問題が

Ising Hamiltonian

として表現できるからです。

例：

組合せ最適化
QAOA
スピンモデル
磁性体解析

---

# QAOAとの関係

QAOAでは：

コスト関数が

Hamiltonian

として表現されます。

例：

```text
H = Z₀Z₁ + Z₁Z₂
```

これが：

最適化対象になります。

---

# VQEとの関係

VQEでは：

次を計算します：

```text
⟨ψ|H|ψ⟩
```

つまり：

Hamiltonianの期待値

です。

そのため：

Pauli分解された形が必要になります。

---

# なぜPauli分解が計算しやすいのか？

理由：

Pauli演算子は

個別に測定できる

からです。

つまり：

```text
H = aX + bZ
```

なら：

⟨X⟩
⟨Z⟩

を別々に測定できます。

---

# 期待値計算の分解

例えば：

```text
H = aX + bZ
```

なら：

```text
⟨H⟩ = a⟨X⟩ + b⟨Z⟩
```

になります。

つまり：

測定可能になります。

---

# Pauli分解と量子回路

重要なポイント：

Pauli文字列は

回路として実装できます。

例：

```text
exp(-iθZ)
```

は：

RZゲート

になります。

また：

```text
exp(-iθX)
```

は：

RXゲート

になります。

---

# 多体系の場合

例えば：

```text
exp(-iθZ₀Z₁)
```

は：

CNOTとRZで実装できます。

つまり：

相互作用Hamiltonianも

回路に変換できます。

---

# Hamiltonian simulationへの準備

Hamiltonian simulationでは：

次を実行します：

```text
exp(-iHt)
```

ここで：

H はPauli分解されています：

```text
H = Σ cᵢPᵢ
```

つまり：

```text
exp(-iHt) = exp(-iΣ cᵢPᵢ t)
```

になります。

これが：

時間発展演算子

です。

---

# 次に必要になる技術

しかし：

問題があります。

一般には：

```text
exp(A + B) ≠ exp(A)exp(B)
```

です。

つまり：

単純には分解できません。

そこで登場するのが：

**Trotter分解**

です。
