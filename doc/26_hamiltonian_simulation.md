---

# なぜ時間発展が重要なのか？

量子力学では：

状態は時間とともに変化します。

その変化は：

Hamiltonian

によって決まります。

つまり：

```text
Hamiltonianが分かれば
時間発展が分かる
```

ということです。

---

# 時間発展演算子とは何か？

量子状態の時間発展は：

次の演算子で表されます：

genui{"math_block_widget_always_prefetch_v2": {"content": "U = e^{-iHt}"}}

ここで：

| 記号 | 意味          |
| -- | ----------- |
| H  | Hamiltonian |
| t  | 時間          |
| U  | 時間発展演算子     |

です。

---

# なぜこの式が重要なのか？

理由：

すべての量子シミュレーションは

この式の実装

だからです。

つまり：

```text
量子状態
↓
時間発展演算子適用
↓
新しい量子状態
```

になります。

---

# Schrödinger方程式との関係

時間発展演算子は：

Schrödinger方程式の解です：

```text
i d|ψ⟩/dt = H|ψ⟩
```

この解が：

時間発展演算子

になります。

---

# Hamiltonian Simulationとは何か？

Hamiltonian Simulationとは：

```text
exp(-iHt)
```

を量子回路として実装すること

です。

つまり：

物理系の時間発展を再現する

アルゴリズムです。

---

# なぜ量子コンピュータに向いているのか？

理由：

量子系は

量子系でしか効率よく再現できない

からです。

例：

| 系のサイズ | 古典計算 | 量子計算 |
| ----- | ---- | ---- |
| 小     | 可能   | 可能   |
| 中     | 困難   | 可能   |
| 大     | 不可能  | 可能   |

---

# 応用分野

Hamiltonian Simulationは：

次の分野で重要です：

```text
量子化学
材料科学
高温超伝導
量子磁性
フォトニクス
```

つまり：

自然そのもののシミュレーション

です 🌍

# QAOAとの関係

実は：

QAOAも時間発展です：

---

# 時間発展の直感的理解

時間発展とは：

状態ベクトルの回転

です。

イメージ：

```text
初期状態
↓
Hamiltonian作用
↓
状態が回転
```

になります。

---

# 固有状態の場合

もし状態が：

Hamiltonianの固有状態なら：

```text
H|ψ⟩ = E|ψ⟩
```

時間発展は：

```text
位相が変わるだけ
```

です。

つまり：

状態は変わりません。

---

# なぜ重要なのか？

理由：

量子アルゴリズムの多くは：

固有値問題

だからです。

例：

```text
VQE
QPE
量子化学
多体系物理
```

です。

---

# 一般状態の場合

固有状態でなければ：

状態は変化します：

```text
|ψ(t)⟩ = exp(-iHt)|ψ(0)⟩
```

つまり：

時間とともに混ざります。

---

# 量子回路として実装するとは？

目標：

```text
exp(-iHt)
```

を

ゲート列へ変換すること

です。

つまり：

```text
数学
↓
回路
```

への変換です。

---

# しかし問題がある

現実のHamiltonianは：

通常：

複数の項の和

です：

```text
H = H₁ + H₂ + H₃ + ...
```

すると：

```text
exp(-i(H₁+H₂)t)
```

は

そのまま分解できません。

---

# 分解できない理由

一般には：

```text
exp(A+B) ≠ exp(A)exp(B)
```

だからです。

つまり：

近似が必要になります。

---

# 解決方法：Trotter分解

ここで登場するのが：

**Trotter分解**

です。

これは：

時間発展を近似する方法

です。

---

# Trotter分解の基本アイデア

近似：

```text
exp(-i(H₁+H₂)t)
≈ exp(-iH₁t)exp(-iH₂t)
```

です。

さらに：

細かく分割すると：

精度が向上します。

---

# なぜ分割すると良くなるのか？

理由：

短時間では：

誤差が小さい

からです。

つまり：

```text
大きな時間発展
↓
小さな時間発展の繰り返し
```

になります。

---

# 実際の回路構築の流れ

処理：

```text
Hamiltonian分解
↓
各項を指数演算子へ変換
↓
量子ゲートへ変換
↓
繰り返し適用
```

です。

---

# 例：Isingモデルの場合

Hamiltonian：

```text
H = Z₁Z₂ + X₁
```

なら：

時間発展：

```text
exp(-iZ₁Z₂t)
exp(-iX₁t)
```

として実装できます。

# Qiskitで時間発展を扱う

Qiskitでは：

Pauli演算子から：

時間発展回路を構築できます：

```python
from qiskit.quantum_info import SparsePauliOp

H = SparsePauliOp.from_list([
    ("ZZ", 1.0),
    ("XI", 0.5)
])
```

これを：

evolution回路へ変換します。
