# doc/01_what_is_hamiltonian.md

# 第01回 Hamiltonianとは何か

## はじめに

本シリーズでは：

量子コンピュータの最も重要な用途である

**量子シミュレーション（Hamiltonian simulation）**

を扱います。

そしてその中心概念が：

**Hamiltonian（ハミルトニアン）**

です。

この言葉は：

量子力学
量子化学
材料科学
量子アルゴリズム

すべてに登場します。

つまり：

Hamiltonianは

**量子計算の共通言語**

です。

---

# なぜHamiltonianが重要なのか？

量子コンピュータの役割は：

量子状態を操作すること

です。

そして：

量子状態の変化を決めるもの

が

Hamiltonian

です。

言い換えると：

Hamiltonianが分かれば

その系の振る舞いが分かります。

---

# 古典力学におけるHamiltonian

Hamiltonianはもともと：

古典力学

で導入された概念です。

意味は：

**系のエネルギー**

です。

例えば：

運動エネルギー
位置エネルギー

の合計として表されます。

つまり：

```text
Hamiltonian = 系の総エネルギー
```

です。

---

# 量子力学におけるHamiltonian

量子力学では：

Hamiltonianは

**演算子**

になります。

つまり：

数ではなく

行列

です。

例：

```text
H = Z
```

これは：

1量子ビット系のHamiltonianです。

---

# なぜ演算子になるのか？

量子力学では：

観測量

はすべて

演算子

として表現されます。

例：

位置
運動量
エネルギー

その中でも：

エネルギー演算子

が

Hamiltonianです。

---

# Hamiltonianが決めるもの

Hamiltonianは：

次の3つを決めます：

① 系のエネルギー
② 固有状態
③ 時間発展

つまり：

量子系のすべて

を決めます。

---

# 固有値としてのエネルギー

量子力学では：

エネルギーは

固有値

として現れます。

つまり：

```text
H |ψ⟩ = E |ψ⟩
```

が成立します。

ここで：

|ψ⟩ は状態
E はエネルギー

です。

---

# 固有状態とは何か？

固有状態とは：

測定しても変化しない状態

です。

つまり：

エネルギー測定をしても

同じ値が出続けます。

この状態が：

量子シミュレーションの目標

になります。

---

# なぜ固有値問題が重要なのか？

量子計算の多くの問題は：

固有値問題

として書けます。

例：

基底状態エネルギー探索
分子構造解析
材料特性推定

これらはすべて：

Hamiltonianの固有値問題

です。

---

# Hamiltonianと量子アルゴリズムの関係

これまで学んだアルゴリズムも：

Hamiltonianと関係しています。

例：

VQE → 最小固有値探索
QAOA → コストHamiltonian最適化
Quantum Kernel → 状態変換構造

つまり：

Hamiltonianはすでに登場していました。

---

# 例：最も簡単なHamiltonian

最小例：

```text
H = Z
```

これは：

次の行列です：

```text
[ 1   0 ]
[ 0  -1 ]
```

このとき：

|0⟩ のエネルギーは +1
|1⟩ のエネルギーは −1

になります。

---

# 複数量子ビットのHamiltonian

複数量子ビットでは：

テンソル積

を使います。

例：

```text
H = Z₀ Z₁
```

これは：

2量子ビット間の相互作用

を表します。

---

# 相互作用とは何か？

量子系では：

粒子同士が影響し合います。

例：

スピン同士
電子同士
原子同士

これを表現するのが：

相互作用Hamiltonian

です。

---

# なぜ相互作用が重要なのか？

理由：

現実の物質は

相互作用でできている

からです。

つまり：

材料
分子
化学反応

すべて：

相互作用Hamiltonian

で記述できます。

---

# Isingモデルの例

最も有名な例：

Isingモデル

です。

例：

```text
H = Z₀Z₁ + Z₁Z₂
```

これは：

隣接スピンの相互作用

を表します。

このモデルは：

磁性体
最適化問題
QAOA

に登場します。

---

# Hamiltonian simulationとは何か？

Hamiltonian simulationとは：

次の操作です：

```text
U(t) = exp(-iHt)
```

つまり：

Hamiltonianに従った

時間発展

を計算します。

これが：

量子シミュレーションの核心

です。

---

# なぜ時間発展が重要なのか？

理由：

自然界は

時間とともに変化する

からです。

例：

電子の移動
化学反応
スピンの振動

これらはすべて：

時間発展として記述できます。

---

# 量子コンピュータの本来の役割

量子コンピュータは：

量子状態を計算する装置

です。

そして：

量子状態の変化は

Hamiltonianが決めます。

つまり：

Hamiltonian simulationこそ

量子計算の本命用途

です。

---

# 開発者視点でのHamiltonian

開発者にとって重要なのは：

Hamiltonianを

行列として理解すること

です。

例えば：

```text
H = aX + bZ + cZZ
```

のように書けます。

これは：

量子回路として実装可能

です。

---

# なぜPauli演算子で書くのか？

理由：

量子回路が

Pauli演算子で構成される

からです。

つまり：

HamiltonianをPauli分解すると

回路に変換できます。
