```text
量子計算は何が解けるのか？

```

---

# 計算量クラスとは何か？

計算量クラスとは：

どの問題がどの程度の計算資源で解けるか

を分類する枠組みです。

例：

| クラス | 意味          |
| --- | ----------- |
| P   | 古典計算で高速に解ける |
| NP  | 解の検証が高速     |
| BPP | 確率的古典計算     |
| BQP | 確率的量子計算     |

です。

---

# BQPとは何か？

BQPとは：

量子コンピュータで

多項式時間内に

高確率で解ける問題の集合

です。

つまり：

```text
Polynomial time
+
Quantum computation
+
Bounded error
```

です。

---

# bounded errorとは何か？

bounded errorとは：

誤答確率が一定以下である

という意味です。

例：

```text
正解確率 ≥ 2/3
```

なら：

BQPに含まれます。

重要なのは：

この確率は繰り返し実行で改善できる

という点です。

---

# なぜ確率が登場するのか？

量子計算は：

測定が確率的

だからです。

つまり：

```text
常に正解
ではなく
高確率で正解
```

が基本モデルになります。

---

# BQPの定義（直感）

ある問題がBQPに属するとは：

次を満たすことです：

```text
量子回路サイズ = polynomial
測定回数 = polynomial
誤答確率 ≤ 定数
```

です。

---

# 古典計算との比較

代表的な関係：

```text
P ⊆ BPP ⊆ BQP
```

と考えられています（ただし未証明部分あり）。

つまり：

量子計算は

確率的古典計算より強い可能性があります。

---

# NPとの関係はどうなっているのか？

重要：

次は未解決問題です：

```text
BQP ⊆ NP ?
NP ⊆ BQP ?
```

どちらも証明されていません。

つまり：

量子計算がNP問題を解けるかは

まだ分かっていません。

---

# よくある誤解

誤解：

```text
量子コンピュータはNP完全問題を高速に解ける
```

これは現在：

証明されていません。

つまり：

```text
量子 = 万能高速計算
ではない
```

です。

---

# Shorアルゴリズムはどこに属するのか？

整数因数分解は：

```text
BQP に含まれる
```

ことが証明されています。

一方：

古典計算では：

```text
P に含まれるか不明
```

です。

つまり：

ここに

量子優位性の典型例があります。

---

# Groverアルゴリズムの位置づけ

Groverは：

探索問題に対して：

```text
O(N) → O(√N)
```

の高速化を実現します。

これは：

BQPの代表例です。

ただし：

指数高速化ではありません。

---

# BPPとは何か？

BPP：

Bounded-error Probabilistic Polynomial time

とは：

確率的古典計算のクラスです。

つまり：

```text
ランダムアルゴリズム
+
多項式時間
+
高確率で正解
```

です。

---

# BPPとBQPの違い

比較：

| クラス | 使用資源    |
| --- | ------- |
| BPP | 古典乱数    |
| BQP | 量子重ね合わせ |

つまり：

量子干渉が追加されます。

---

# 量子干渉の役割

量子計算の本質：

```text
不要な解を打ち消す
必要な解を強める
```

です。

これにより：

古典より強い計算が可能になります。

---

# なぜBQPが重要なのか？

理由：

量子計算の限界を示す

からです。

つまり：

```text
量子で解ける問題
```

の境界になります。

---

# BQPに含まれる代表問題

代表例：

```text
整数因数分解
離散対数問題
量子シミュレーション
ランダム回路サンプリング
```

です。

これらは：

古典計算では困難です。

---

# BQPに含まれないと考えられている問題

例：

```text
NP完全問題
```

です。

代表例：

```text
SAT
巡回セールスマン問題
グラフ彩色問題
```

などです。

ただし：

証明はありません。

---

# NP完全問題との関係

現在の理解：

```text
NP完全 ⊄ BQP
```

と考えられています（未証明）。

つまり：

量子計算でも

万能ではありません。

---

# PostBQPとは何か？

興味深いクラス：

PostBQP

があります。

これは：

測定結果を条件付きで選べる

仮想モデルです。

結果：

```text
PostBQP = PP
```

が証明されています。

これは：

非常に強力な計算モデルです。

---

# なぜPostBQPが重要なのか？

理由：

量子計算の理論的潜在能力

を示すからです。

つまり：

```text
測定制約が
量子計算の限界を作っている
```

ことが分かります。

---

# BQP完全問題とは何か？

古典計算では：

NP完全問題

があります。

同様に：

BQP完全問題

も研究されています。

代表例：

```text
Local Hamiltonian問題
```

です。

---

# Local Hamiltonian問題とは何か？

これは：

量子版SAT問題

です。

つまり：

```text
量子状態の基底エネルギー計算
```

問題です。

量子化学と密接に関係します。

---

# なぜ重要なのか？

理由：

VQEの理論基盤

だからです。

つまり：

```text
VQE = Local Hamiltonian problem solver
```

と考えられます。

---

# BQPの直感的理解

まとめると：

BQPとは：

```text
量子干渉を使って
多項式時間で
高確率に解ける問題の集合
```

です。

---

# 開発者視点での意味

重要：

BQPを理解すると：

次が分かります：

```text
どの問題に量子を使うべきか
```

です。

つまり：

量子アルゴリズム選択の基準になります。

---

# Quantum Advantageとの関係

Quantum Advantageとは：

次が成立する領域です：

```text
問題 ∈ BQP
かつ
問題 ∉ BPP
```

です。

ここが：

量子計算の価値領域です。
